# Elevate Website

## Project Overview

Elevate is a premium AI-powered marketing agency.

The website should feel:
- Modern
- Premium
- Clean
- Minimal
- Professional
- High-end

The design should match the Figma file as closely as possible.

---

## Tech Stack

- React
- Vite
- Tailwind CSS v4

---

## Design Principles

- Pixel-perfect implementation.
- Responsive across desktop, tablet, and mobile.
- Smooth animations.
- Reusable components.
- Semantic HTML.
- Accessible markup.

---

## Assets
